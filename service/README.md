## Description

[Nest](https://github.com/nestjs/nest) The service was bootstrapped using the standard Nestjs TypeScript starter repository.

## Dev Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
